## 프로젝트 배경

What is this for?
* 악성 광고 실시간 검열 차단
* 맞춤형 서비스를 위한 고객 정보 처리


### Built With
* [C#]
* [Selenium](https://www.selenium.dev/)
* [AutoIT](https://www.autoitscript.com/site/)

### 구현 상황
* 엑셀 연동
* 포스팅 내용중 광고성 메세지 포함여부 검사
* 자체검열 등 예외처리 로직
* 악성 광고 패턴 분석
* 관리자 계정으로 자동 로그인 & 로그아웃
